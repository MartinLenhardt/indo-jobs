# Faire Wege für Pflegekräfte aus Indien nach Deutschland

Die Indo-Jobs GmbH vermittelt qualifizierte Pflegekräfte aus Indien an deutsche Pflegeeinrichtungen und Krankenhäuser.

Unser Ziel ist ein faires, transparentes und nachhaltiges Recruiting.

## Unsere Leistungen
- Rekrutierung qualifizierter Pflegekräfte in Indien
- Deutschsprachkurse bis B2
- Unterstützung im Visaverfahren
- Begleitung im Anerkennungsverfahren
- Integration und Unterbringung in Deutschland
