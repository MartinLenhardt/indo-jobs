# Ausbildung in Indien

In Zusammenarbeit mit der Theresienakademie in Kerala werden Pflegekräfte gezielt auf ihre berufliche Zukunft in Deutschland vorbereitet.

Die Ausbildung umfasst:
- Deutschsprachkurse bis B2
- interkulturelle Vorbereitung
- Vorbereitung auf das deutsche Gesundheitssystem

Die Sprachprüfungen erfolgen über das Goethe-Institut in Kochi.
