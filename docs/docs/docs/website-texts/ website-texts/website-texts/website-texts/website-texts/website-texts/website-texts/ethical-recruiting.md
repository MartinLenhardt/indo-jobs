# Ethical Recruiting

Die Indo-Jobs GmbH verpflichtet sich zu einem fairen und verantwortungsvollen Recruiting internationaler Pflegekräfte.

## Grundsätze
- keine Vermittlungsgebühren für Pflegekräfte
- transparente Arbeitsverträge
- qualifizierte sprachliche Vorbereitung
- faire Arbeitsbedingungen
- Unterstützung bei Integration und Anerkennung
