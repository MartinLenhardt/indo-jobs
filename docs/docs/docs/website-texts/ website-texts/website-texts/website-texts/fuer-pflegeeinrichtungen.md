# Für Pflegeeinrichtungen

Indo-Jobs unterstützt Pflegeheime, Krankenhäuser und ambulante Dienste bei der nachhaltigen Gewinnung qualifizierter Pflegefachkräfte aus Indien.

## Leistungen
- Vorauswahl qualifizierter Pflegekräfte
- Organisation von Sprachkursen bis B2
- Unterstützung im Visaverfahren
- Begleitung bei Anerkennungsverfahren
- Unterstützung bei Integration und Unterkunft
