# Code of Conduct für Partner-Pflegeeinrichtungen

Partner verpflichten sich zu:
- Einhaltung des deutschen Arbeitsrechts
- Fairer Vergütung
- Gleichbehandlung internationaler Pflegekräfte
- Unterstützung der fachlichen und sozialen Integration
- Unterstützung im Anerkennungsverfahren
- Respektvoller und diskriminierungsfreier Zusammenarbeit
