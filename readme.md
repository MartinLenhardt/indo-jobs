# Indo-Jobs Website

Projektdateien für die Website der Indo-Jobs GmbH.

## Inhalt
- WordPress-Importdatei
- Logo
- Rechtstexte
- Website-Texte

## Struktur
- `wordpress/` → Importdatei
- `assets/logo/` → Logo
- `docs/` → AGB, Datenschutz, Impressum etc.
- `website-texts/` → Inhalte der Website

## Verwendung
### WordPress
Die Datei `wordpress/indojobs-wordpress-import.xml` kann in WordPress importiert werden über:

Tools → Import → WordPress

### Rechtstexte
Die Dateien im Ordner `docs/` können direkt in WordPress-Seiten übernommen werden.

### Website-Inhalte
Die Dateien im Ordner `website-texts/` enthalten die Texte für die einzelnen Unterseiten.
